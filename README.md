# Stakeverse

Stakeverse is a betting command center focused on controlled staking, bankroll discipline, source tracking, ROI analytics, and future AI-assisted match analysis.

## Stack

- Next.js
- Supabase
- Vercel

## Environment

Copy `.env.example` to `.env.local` and fill:

```env
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_PUBLISHABLE_KEY=
```

## Database

Run `supabase/schema.sql` in the Supabase SQL Editor.
